# Express-Postgres-Auth-Api
An authentication api with exprss and postgress

```
sudo systemctl start postgresql.service
```
```
Step 1
add .env file and add conent 
DB_USER=
DB_PASSWORD=
DB_HOST=
DB_PORT=
DB_DATABASE=auth_db
JWT_KEY=
SENDGRID_USERNAME=
SENDGRID_PASSWORD=

Step 2
yarn install

Step 3 
yarn start
```
